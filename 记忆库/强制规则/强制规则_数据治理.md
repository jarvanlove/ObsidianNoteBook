## 强制规则：数据治理

### 归档原则

**禁止删除，只归档。**

- 不需要的记忆 → 移入 `记忆库归档/` 目录
- 不需要的任务 → 移到 `✅ 已完成` 或 `任务归档/`
- 过时的规则 → 标记 `[已废弃]` 而非删除

### 隐私保护

在记录任何内容到记忆库前，检查：

1. **是否包含密钥/Token？**
   - API Key → 禁止存储，使用环境变量
   - 密码 → 禁止存储
   - Token → 禁止存储

2. **是否包含个人身份信息？**
   - 身份证号、银行卡号 → 禁止存储
   - 真实姓名+敏感信息 → 脱敏或禁止存储

3. **是否包含商业机密？**
   - 客户数据 → 检查合规要求
   - 内部定价 → 谨慎处理

### 脱敏标准

当必须记录包含敏感词的内容时：
- 替换为 `[KEY]`、`[TOKEN]`、`[姓名]` 等占位符
- 在文件顶部添加 `> ⚠️ 本文件已脱敏，请勿包含真实密钥`

### 数据保留

| 类型 | 保留期限 | 归档位置 |
|------|---------|---------|
| 情景记忆 | 永久 | 记忆库/情景记忆/ |
| 语义记忆 | 永久 | 记忆库/语义记忆/ |
| 强制规则 | 永久（废弃标记） | 记忆库/强制规则/ |
| 任务清单 | 当前有效 | 任务清单.md |
| 已完成任务 | 12个月 | 任务归档/ |
