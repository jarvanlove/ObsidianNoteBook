## 强制规则：任务与想法追踪

### 任务识别触发词

当用户说出以下词汇时，必须将其加入任务清单：

| 触发词 | 示例 | 自动分类 |
|--------|------|---------|
| "要做" | "明天要做XX" | 🟡 重要不紧急 |
| "必须" | "必须完成YY" | 🔴 紧急重要 |
| "记得" | "记得做ZZ" | 🟡 重要不紧急 |
| "完成了" | "完成了AA" | ✅ 已完成 |
| "截止" | "截止日期是..." | 🔴 紧急重要 |

### 想法捕捉

当用户表达了一个想法但不紧急时：
1. 先确认："是否需要加入任务清单？"
2. 如果是，添加到 🟢 日常事项
3. 如果否，在情景记忆中记录这个想法

### 任务完成流程

1. 用户说"完成了XX"
2. 从原位置删除该任务
3. 追加到 `任务清单.md` 的 `✅ 已完成` 区
4. 可选：写一条情景记忆记录完成情况

### 定期提醒

- 每周一：提醒用户 review 任务清单
- 每月末：归档已完成任务到 `任务归档/YYYY-MM.md`
