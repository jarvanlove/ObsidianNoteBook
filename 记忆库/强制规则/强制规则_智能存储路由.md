## 强制规则：智能存储路由

### 路由决策树

```
收到用户内容或文件
    │
    ▼
是 Claude Code CLI 的输出/产物吗？
    │
    ├─ YES → 放在当前工作目录（项目目录）
    │
    └─ NO
        │
        ▼
    是 AI 推理过程、中间思考吗？
        │
        ├─ YES → 不存储，内存中处理
        │
        └─ NO
            │
            ▼
        应该长期保存的知识/经验？
            │
            ├─ YES → 写入记忆库（见下）
            │
            └─ NO → 放在工作目录或丢弃
```

### 记忆库路由

| 内容类型 | 写入位置 | 命名格式 |
|---------|---------|---------|
| 对话事件 | `记忆库/情景记忆/YYYY-MM/` | `MMDD-事件主题.md` |
| 知识提炼 | `记忆库/语义记忆/` | `主题名.md` |
| 行为约束 | `记忆库/强制规则/` | `强制规则_名称.md` |
| 任务 | `任务清单.md` | 按优先级分区 |

### 禁止

- 禁止在项目目录创建"我的记忆库"等个人笔记
- 禁止在项目目录创建零散的 `.md` 个人笔记文件
- 所有个人知识必须经过路由进入记忆库
