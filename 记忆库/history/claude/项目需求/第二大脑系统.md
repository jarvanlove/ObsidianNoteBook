# 第二大脑系统

**基于Claude Code + Obsidian的跨平台第二大脑NPM包系统**

📁 路径：`C:\Work\note\ObsidianNoteBook`

🔧 功能：
- 三层记忆架构(情景记忆/语义记忆/强制规则)
- v2.0自动项目路由(根据工作目录识别项目打标签)
- 任务追踪与时间意图捕获
- 外部知识吸收6步流程
- NPM CLI工具(@jiawenwu/secondbrain)
- 支持SECONDBRAIN_PATH环境变量自定义记忆库路径
