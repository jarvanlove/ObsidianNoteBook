## 1. 产品概述

### 1.1 产品定位

**科研智能体**是一个基于大语言模型的通用科研助手，帮助高校科研人员通过自然语言完成各类科研任务。

**核心定位**：让科研人员拥有一个"会思考、能反思、记得住"的AI伙伴。

### 1.2 目标用户

| 用户角色 | 核心需求 |
|---------|---------|
| **科研人员** | 文献综述、论文写作、数据分析、申报书生成 |
| **高校校长/管理层** | 科研进度看板、项目风险预警、成果统计 |
| **行政人员** | 红头文件解析、科技报告生成 |

### 1.3 核心价值

```
Dify现状                    科研智能体
─────────────────────────────────────────────────────────
"帮我查文献" → 打开知识库    "帮我综述这个主题" → 直接搞定
"第三部分重写" → 重新生成一次  "第三部分重写" → 反思修复直到满意
上次说了什么来着...         "记得我上周在做的项目吗" → 当然记得
```

## 3. 功能需求

### 3.1 核心功能

#### F1：通用科研对话框

**描述**：用户通过自然语言与AI交互，完成各类科研任务

**功能点**：
- 自然语言理解，自动分解任务
- 流式输出，实时显示生成进度
- 多轮对话，支持上下文记忆
- 反思机制，产出质量可控

**用户操作**：
1. 输入自然语言描述需求
2. 查看流式生成的输出
3. 如不满意，输入修改意见
4. 满意后确认产出

**验收标准**：
- [ ] 支持自然语言输入
- [ ] 流式输出延迟 < 1秒
- [ ] 反思修复轮次可达3次
- [ ] 上下文窗口支持50轮对话

#### F2：文献综述Agent

**描述**：自动完成文献搜索、爬取、摘要、综述生成

**功能点**：
- 支持关键词、主题搜索
- 自动爬取论文详情（arXiv、PubMed等）
- 智能摘要提取
- 自动生成综述报告
- 支持指定参考文献格式

**用户操作**：
1. 输入综述主题和范围
2. 选择搜索来源
3. 确认综述大纲
4. 查看生成结果
5. 提出修改意见或确认同步

**验收标准**：
- [ ] 支持至少3个学术搜索源
- [ ] 单次综述最多50篇文献
- [ ] 生成时间 < 5分钟
- [ ] 支持导出Word/PDF

#### F3：论文写作Agent

**描述**：辅助论文各章节写作

**功能点**：
- 支持Introduction、Method、Result等章节
- 自动关联文献
- 格式排版控制
- 语法检查
- 参考文献管理

**验收标准**：
- [ ] 支持至少5种期刊格式模板
- [ ] 自动引用文献
- [ ] 语法检查准确率 > 90%

#### F4：数据分析Agent

**描述**：数据清洗、统计分析、可视化

**功能点**：
- 数据文件上传（CSV、Excel）
- 自动数据清洗
- 统计分析（t检验、方差分析等）
- 自动图表生成
- 结果解读

**验收标准**：
- [ ] 支持CSV/Excel文件
- [ ] 常用统计方法全覆盖
- [ ] 图表可导出
- [ ] AI可解释统计结果

#### F5：成果同步

**描述**：AI产出同步到科研平台项目里程碑

**功能点**：
- 选择目标项目
- 选择目标里程碑
- 自动创建成果记录
- 支持附件上传

**验收标准**：
- [ ] 可选择已有项目
- [ ] 自动创建里程碑记录
- [ ] 支持附件

#### F6：反思机制

**描述**：AI产出后自动评审，不满意则重新生成

**功能点**：
- Critic Agent自动评审
- 返回修改建议
- 支持多轮循环
- 达到轮次上限时强制输出

**验收标准**：
- [ ] Critic评审准确率 > 80%
- [ ] 最大循环3轮
- [ ] 用户可跳过评审直接确认

### 3.2 平台集成功能

#### F7：MCP-Platform封装

**描述**：把科研平台Java API封装为MCP工具

**封装范围**：
- 项目管理API
- 里程碑API
- 用户管理API
- 文件管理API
- 仪表盘API

**验收标准**：
- [ ] 覆盖现有Java API 80%以上
- [ ] 支持自然语言调用
- [ ] 调用延迟 < 2秒

#### F8：Dify知识库集成

**描述**：通过MCP封装复用现有Dify知识库

**功能点**：
- 直接调用Dify知识库API
- 不迁移数据
- 新旧系统并行

**验收标准**：
- [ ] 支持知识库检索
- [ ] 返回结果格式统一
