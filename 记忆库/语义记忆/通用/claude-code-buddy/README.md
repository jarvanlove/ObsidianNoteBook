# claude-code-buddy 知识库 README

## 这是什么

这是一个围绕 `Claude Code /buddy` 命令整理出来的 Obsidian 小型知识库。

目标不是把零碎观察堆成一篇长笔记，而是把 `/buddy` 这个功能拆成几个稳定问题：

1. `/buddy` 到底是什么，解决了什么问题。
2. 这个伴侣系统是怎么生成、存储、渲染和触发的。
3. 为什么它虽然是“轻功能”，却体现了 Claude Code 的产品方法。
4. 如果以后你要做自己的 agent 产品，这套机制可以迁移什么，不能迁移什么。

---

## 当前分类方式

当前目录按“进入 -> 理解 -> 拆解 -> 启发”四层组织：

- `00-入口`
  - 面向快速进入和 Obsidian 阅读导航。
- `01-命令理解`
  - 面向 `/buddy` 命令本身、能力边界、生成逻辑。
- `02-交互机制`
  - 面向 UI 渲染、状态存储、行为触发。
- `03-产品启发`
  - 面向产品设计抽象与迁移方法。
