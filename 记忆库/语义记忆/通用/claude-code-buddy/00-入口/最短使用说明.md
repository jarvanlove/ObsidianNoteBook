## 一句话定义

`/buddy` 不是提升编码产能的主命令，而是 Claude Code 在终端中植入的一层“轻陪伴界面”。

它给用户一个可持续存在、可被识别、可被轻交互的 ASCII 伴侣，用极低成本提升会话的温度、记忆点和产品辨识度。

---

## 你真正需要知道的 5 件事

1. `/buddy` 的本质不是工具增强，而是体验增强。
2. 它不是一次性玩笑，而是一套带有生成、渲染、状态存储、交互反馈的完整系统。
3. 伴侣不是完全随机，而是基于用户身份稳定生成，所以用户会觉得“这是我的那个 buddy”。
4. 用户能做的显性交互很少，比如 `/buddy pet`，但这反而保证了它不抢主流程。
5. 这个功能最值得研究的地方，不是宠物本身，而是 Claude Code 如何在高强度工程工具里插入人格化层。
