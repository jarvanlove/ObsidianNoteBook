## 使用说明

### 记录与任务

| 你说的话 | Claude 做什么 |
|---------|-------------|
| `启动第二大脑` | 读任务清单，显示今日概览 |
| `必须做XX` | 加入 🔴 紧急重要 |
| `明天要做XX` | 加入 🟡 重要不紧急 |
| `有空时做XX` | 加入 🟢 日常事项 |
| `记录这个决策` | 写情景记忆（自动打项目标签） |
| `总结成经验` | 写语义记忆（自动打项目标签） |
| `这是通用知识` | 写语义记忆到 `通用/`（无项目标签） |
| `看看这篇文章+链接` | 启动 6 步知识吸收 |
| `完成了XX` | 移到 ✅ 已完成 |

### 查询

| 你说的话 | Claude 做什么 |
|---------|-------------|
| `查看#某项目的记忆` | 筛选该项目所有情景+语义记忆 |
| `看看有什么任务` | 显示任务清单 |

### 自动行为

- 检测到时间意图 → 自动添加任务
- 每周日 → 提醒归档已完成任务
- 任务过期 → 提醒并询问是否调整优先级

### GitHub 提交规则

- 触发词：`commit` / `提交` / `/commit`（被动，等你指令）
- 禁止自动提交（FATAL-005）
- 只提交：`记忆库/`（含任务清单、搭建指南、源文）

### 语义记忆分类

| 内容类型       | 写入位置           |
| ---------- | -------------- |
| 关于第二大脑系统本身 | `语义记忆/第二大脑/`   |
| 跨项目通用知识    | `语义记忆/通用/`     |
| 某项目专属经验    | `语义记忆/<该项目名>/` |
